# Donghua VF Player V3

Version 3 de l'application avec :

- design premium inspiré des plateformes de streaming ;
- recherche instantanée ;
- filtres ;
- tri par date, titre ou épisode ;
- favoris sauvegardés ;
- statistiques ;
- bibliothèque personnelle ;
- lecteur vidéo ;
- interface optimisée pour iPhone.

Les vidéos doivent provenir de liens que l'utilisateur a légalement le droit d'utiliser.
