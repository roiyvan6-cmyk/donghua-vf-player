# Donghua VF Player V4

Fonctionnalités :

- logo demi-lune mauve avec 伊凡 ;
- intégration de vidéos YouTube dans le site ;
- intégration de playlists YouTube ;
- liens vidéo directs MP4/WebM ;
- import temporaire de fichiers vidéo depuis l’iPhone ;
- favoris ;
- recherche et filtres ;
- progression de lecture pour les vidéos directes ;
- section Continuer à regarder ;
- statistiques ;
- interface mobile.

## Limites importantes

- Une vidéo YouTube peut refuser l’intégration si son propriétaire l’a désactivée.
- Les commandes et l’identité visuelle YouTube restent visibles dans le lecteur officiel.
- Un fichier local choisi depuis l’iPhone n’est pas envoyé sur GitHub et peut devoir être sélectionné à nouveau après fermeture de Safari.
- Cette application n’extrait ni ne télécharge les vidéos depuis les plateformes.
- Utiliser uniquement des contenus que vous êtes autorisé à regarder ou intégrer.
