# Donghua VF Player V4.1

Améliorations principales :

- champ « Épisode(s) / partie » en texte libre : `1`, `1 à 5`, `Film`, `OVA 2`, etc. ;
- couverture importable depuis l’iPhone ou par URL ;
- compression automatique de la couverture pour économiser le stockage du navigateur ;
- intégration YouTube avec interface française et sous-titres français demandés par défaut ;
- ajout d’une URL d’intégration officielle provenant d’une autre plateforme ;
- fichiers vidéo locaux et liens vidéo directs ;
- ajout facultatif de sous-titres personnels WebVTT (`.vtt`) aux vidéos directes/locales ;
- favoris, recherche, progression, filtres et statistiques.

## Limites

- YouTube choisit réellement la piste disponible : le français n’est pas garanti si aucune piste française ou traduction compatible n’est fournie.
- L’application ne peut pas masquer ou supprimer des sous-titres incrustés directement dans l’image d’une vidéo.
- Les sous-titres personnels `.vtt` ne peuvent pas être superposés dans le lecteur YouTube intégré.
- Une plateforme externe peut interdire son affichage dans un iframe.
- L’application ne télécharge et n’extrait aucune vidéo d’une plateforme.
